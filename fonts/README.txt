STUDIO RAGE — LOCAL FONT SETUP

The site never requests fonts from an external CDN.

Preferred families:
- Helvetica Neue: primary interface, body and titles. The CSS first checks whether it is installed locally.
- PP Editorial New: limited editorial accent in the footer/contact call-to-action.
- Oxygen Mono: service lists, navigation labels and metadata.

For a fully identical appearance on every device, place legally licensed WOFF2 files in this folder and add their file URLs to css/vivid-refinement.css. Font files are intentionally not included in this archive.
